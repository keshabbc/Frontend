# 🌐 Frontend Development Journey

Welcome to my **Frontend Development Journey** repository! 🚀

This repository documents my learning path from the fundamentals of web development to building modern, responsive, and interactive web applications. Every topic is organized into folders with notes, examples, and practice exercises.

Each lesson contains its own **`index.html`**, making it easy to preview and deploy every topic individually using GitHub Pages or any static hosting platform.

# 🚀 Deployment

This repository is designed to work seamlessly with **GitHub Pages**, **Netlify**, or **Vercel**.

Each topic contains its own `index.html`, allowing direct access after deployment.

Example:

```text
https://username.github.io/frontend-learning/
https://username.github.io/frontend-learning/A-HTML/01-Introduction/
https://username.github.io/frontend-learning/B-CSS/09-Flexbox/
https://username.github.io/frontend-learning/C-JavaScript/10-DOM/
```

The root `index.html` acts as the homepage for navigating all lessons.
